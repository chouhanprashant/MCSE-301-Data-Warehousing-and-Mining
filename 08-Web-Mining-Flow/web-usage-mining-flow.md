# Web Usage Mining – Very Simple Explanation (Exam-Pass Level)

## Definition
Web Usage Mining ek web mining technique hai jo
**user ke behaviour** ko analyse karti hai.

Simple words:
> Web Usage Mining batata hai user website par kya karta hai.

---

## What Data is Used?
Web Usage Mining mainly use karti hai:
- Server logs
- Click data
- Page visit records
- User session data

---

## Why Web Usage Mining?
Website owner ko ye jaana hota hai:
- User kaun si page visit karta hai
- Kitna time spend karta hai
- Kis page se exit karta hai

Isse website improve hoti hai.

---

## Main Steps of Web Usage Mining

### 1️⃣ Data Collection
- Web server logs collect kiye jaate hain

---

### 2️⃣ Data Preprocessing
- Duplicate entries remove
- Irrelevant data hataana
- User sessions identify karna

---

### 3️⃣ Pattern Discovery
- User behaviour patterns find kiye jaate hain
- Frequent paths, clicks analyse hote hain

---

### 4️⃣ Pattern Analysis
- Useful patterns select kiye jaate hain
- Decision making ke liye use hote hain

---

## Simple Flow
Web Logs  
↓  
Preprocessing  
↓  
Pattern Discovery  
↓  
Analysis  
↓  
Useful Information  

---

## Applications of Web Usage Mining
- Website personalization
- Recommendation systems
- Web traffic analysis
- Website performance improvement

---

## Advantages
- User behaviour samajh aata hai
- Website design improve hoti hai
- Business decisions better hote hain

---

## Limitations
- Privacy issues ho sakte hain
- Web data noisy hota hai

---

## Exam Trigger Line
Web Usage Mining analyzes user behavior using web log data to discover usage patterns.
