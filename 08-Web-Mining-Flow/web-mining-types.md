# Web Mining – Types (Very Simple, Exam-Pass Level)

## What is Web Mining?
Web Mining ek data mining technique hai jo
**web par available data** se useful information nikalti hai.

Simple words:
> Web Mining website ke data ko analyse karke patterns nikalti hai.

---

## Types of Web Mining

Web Mining mainly **3 types** ki hoti hai.

---

### 1️⃣ Web Content Mining
Isme:
- Web pages ke **content** analyse kiye jaate hain
- Text, images, audio, video ka use hota hai

Example:
- News website ke articles analyse karna
- Blog content se information nikalna

---

### 2️⃣ Web Structure Mining
Isme:
- Website ke **links aur structure** analyse kiye jaate hain
- Kaunsa page kis page se linked hai, ye dekha jaata hai

Example:
- Search engine ranking
- Page importance nikalna

---

### 3️⃣ Web Usage Mining
Isme:
- **User behaviour** analyse kiya jaata hai
- Server logs, click data use hota hai

Example:
- User ka click pattern
- Recommendation systems

---

## Simple Flow of Web Mining
Web Data  
↓  
Data Preprocessing  
↓  
Pattern Discovery  
↓  
Useful Information  

---

## Applications of Web Mining
- Recommendation systems
- Website personalization
- Customer behaviour analysis
- Web traffic analysis

---

## Advantages
- User experience improve hota hai
- Business decision better hote hain
- Website performance improve hoti hai

---

## Limitations
- Web data noisy hota hai
- Privacy issues ho sakte hain

---

## One-Line Understanding
Web Mining web data se useful patterns aur information nikalne ki process hai.

---

## Exam Trigger Line
Web Mining is the application of data mining techniques to discover patterns from web data.
